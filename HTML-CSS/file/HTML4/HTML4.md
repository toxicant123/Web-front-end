### 2. 计算机基础知识

1. 计算机俗称电脑，是现代一种用于高速计算的电子计算机器，可以进行数值计算、逻辑计算，还具有存储记忆功能。
2. 计算机由`硬件 + 软件`组成：
   硬件：看得见摸得着的物理部件。
   软件：可以指挥硬件工作的指令。
3. 软件的分类：
   1. 系统软件：Windows、Linux、Android、Harmony 等。
   2. 应用软件：微信、QQ、王者荣耀、PhotoShop 等。
